# ☕ Starbucks Web Clone (Vanilla JS)

HTML, CSS, JavaScript만 사용하여 구현한  
**Starbucks 공식 웹사이트 프론트엔드 클론 프로젝트**입니다.

프레임워크 없이 순수 웹 기술로 레이아웃 구성, 인터랙션, 스타일링을 구현했습니다.

---

## 📌 프로젝트 개요

- Starbucks 공식 홈페이지 UI 분석 후 클론 구현
- Vanilla JavaScript 기반 인터랙션 처리
- 순수 CSS로 레이아웃 및 스타일 구성
- 웹 퍼블리싱 및 구조 설계 연습 목적

---

## 🛠 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📁 프로젝트 구조

StarbucksWeb_FE_Clone  
- index.html        : 메인 페이지  
- css/              : 스타일 파일  
- js/               : JavaScript 파일  
- images/           : 이미지 리소스  

---

## 🚀 실행 방법

1. 프로젝트 클론

   git clone https://github.com/Jeonsowon/StarbucksWeb_FE_Clone.git

2. index.html 파일을 브라우저에서 실행

   또는 VSCode Live Server 사용
